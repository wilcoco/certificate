# 사내 문서 발급 포털

모바일 우선 UI로 사번 로그인 후 재직 증명서, 경력 증명서 등을 내려받을 수 있는 데모 서비스입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000` 접속

## 데모 사번

- 1001 (김하늘)
- 1002 (이준서)
- 1003 (박민지)

## Railway 배포

1. GitHub에 푸시 후 Railway에서 새 프로젝트 생성
2. `NPM` 환경을 선택
3. `SESSION_SECRET` 환경 변수를 설정
4. Deploy 버튼 클릭
